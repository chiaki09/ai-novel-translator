# AI Web Novel Translator - ブラウザ拡張機能

## プロジェクト概要

海外のWeb小説をポップアップボタン1クリックでAI翻訳するChrome拡張機能。
ユーザーが自分のGemini APIキー（Google AI Studio無料枠）を使う**BYOK方式**。

---

## 技術スタック

- Chrome拡張機能 Manifest V3
- Vanilla JS（フレームワークなし、シンプルに）
- Gemini 2.5 Flash API（google ai studioの無料キー）
- chrome.storage.local（キャッシュ・設定保存）

---

## 機能要件

### MVP（最初に作るもの）

**翻訳フロー**
1. ユーザーが海外小説ページを開く
2. 拡張機能のポップアップを開く
3. 「翻訳する」ボタンを押す
4. ページ本文のテキストを段落単位で抽出
5. Gemini APIに投げて日本語訳を取得
6. 元のDOMの文字列を訳文に置き換える
7. 「原文に戻す」ボタンで元に戻せる

**設定画面（options）**
- Gemini APIキーの入力・保存
- APIキーの有効性チェック（テストボタン）

**ポップアップUI**
- 翻訳ボタン（翻訳中はローディング表示）
- 翻訳済みページではトグルで原文/訳文切り替え
- 設定画面へのリンク

**キャッシュ**
- 同一URLの翻訳結果はローカルに保存
- 再訪時はAPIを叩かずキャッシュを使用

### 翻訳対象の抽出ルール（汎用モード）

優先的に翻訳するセレクタ（この順で探してヒットしたものを対象にする）:
```
article, .chapter-content, .content, main p, body p
```
- pタグ単位で翻訳（1リクエスト = 複数段落をまとめて送る）
- ナビゲーション・ヘッダー・フッター・広告は除外

### 翻訳プロンプト

```
システムプロンプト:
あなたはWeb小説の翻訳者です。以下のルールに従って日本語に翻訳してください。
- 自然で読みやすい日本語にする（直訳NG）
- キャラクターの口調・個性を維持する
- 人名・地名・スキル名はカタカナ表記を基本とする
- 段落構造をそのまま維持する
- 翻訳結果のみ返す（説明文・コメント不要）
- 各段落は元と同じ順序・数で返す

ユーザーメッセージ:
以下のテキストを翻訳してください：
[テキスト]
```

## 非機能要件

| 項目 | 内容 |
|---|---|
| APIキー保存 | chrome.storage.localに保存（平文、ユーザーのローカルのみ） |
| エラー処理 | APIエラー時はポップアップにエラーメッセージ表示 |
| レート制限 | 段落を一括送信してリクエスト数を最小化 |
| キャッシュ有効期限 | 設けない（手動クリアのみ） |

## Gemini APIキー取得手順（ユーザー向け）

1. https://aistudio.google.com にアクセス
2. 「Get API Key」→「Create API Key」
3. 取得したキーを拡張機能の設定画面に貼り付け

---

## ローカル開発・動作確認

1. `chrome://extensions` を開く
2. 「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」→ `extension/` フォルダを選択
4. 任意の英語小説ページで動作確認
