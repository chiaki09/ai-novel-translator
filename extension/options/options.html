📋 Gemini APIキーの取得方法

1. Google AI Studio にアクセス

2. 「Get API Key」→「Create API Key」をクリック

3. 取得したAPIキーを下記に入力して保存

APIキーはお使いのブラウザにのみ保存され、外部に送信されません

🔧 トラブルシューティング

❌ "API key not valid" エラーの場合

  1. Google AI Studio で新しいAPIキーを生成
  2. プロジェクトでGemini APIが有効化されていることを確認
  3. APIキーの使用制限・クォータを確認
  4. 古いAPIキーが無効化されていないか確認

🌍 地域制限・利用可能性の確認

  1. Gemini API地域可用性を確認
  2. VPN使用時は米国・EU地域に設定
  3. Google Cloud Consoleでプロジェクト設定を確認

💳 支払い・クォータの確認

  1. Google Cloud 請求でアカウント状況を確認
  2. クォータページでGemini API制限を確認
  3. 無料枠を超過していないか確認
